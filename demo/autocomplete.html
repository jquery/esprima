Editing Autocomplete makes coding tasks fun again

In the following code editor (based on Eclipse Orion), autocompletion is provided by a type inferencer (part of Scripted's Content Assist) which supports both static code analysis and JSDoc-style type annotation.

var capitalDb = {
    Indonesia: 'Jakarta',
    Germany: 'Berlin',
    Norway: 'Oslo'
};

var constants = [3.1415926535, 2.7182818284];

// Property completion: type "capitalDb." and press Ctrl+Space.


// Array inferencing: type "constants[0]." and press Ctrl+Space
// to list possible number conversion (e.g "toFixed").



/**
 * Find the greatest common divisor of two numbers.
 * @param {number} a
 * @param {number} b
 * @return {number}
 */
function gcd(a, b) {
    return (b === 0) ? a : gcd(b, a % b);
}

// Type "gc" and press Ctrl+Space to show the function signature.
