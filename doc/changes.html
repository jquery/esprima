Version 1.0

In this initial release, the syntax tree is very similar to the one produced by Mozilla SpiderMonkey, with the following variants:

Range-based location for each syntax node is available by specifying the range property to true in the options object. Example:

esprima.parse('answer = 42', { range: true });

Raw value of constant literal is available by specifying the raw property to true in the options object. Example:

esprima.parse('answer = 42', { raw: true });

The list of all comments (block and line) is supplied (as an array) if the comment property in the options object is set to true. Example:

esprima.parse('/* Hello */ 42', { comment: true });
